# Task 1: Restaurant Rating Prediction

## 🎯 Objective
The goal of this task is to build a machine learning model that accurately predicts the aggregate rating of a restaurant based on its features.

## 📈 Steps Performed
1. **Data Preprocessing**: Handled missing values, encoded categorical variables (like `Has Table Booking`, `Has Online Delivery`), and split the data into training and testing sets.
2. **Model Selection**: Trained regression models including Linear Regression, Decision Trees, and Random Forests.
3. **Evaluation**: Compared models using Mean Squared Error (MSE) and R-squared ($R^2$) scores to evaluate predictive performance.
