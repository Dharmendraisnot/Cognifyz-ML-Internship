# Task 3: Cuisine Classification

## 🎯 Objective
The goal of this task is to develop a classification model that predicts the type of cuisine a restaurant serves based on features such as location, price range, and user ratings.

## 📈 Steps Performed
1. **Target Engineering**: Identified the most frequent cuisines and preprocessed the text data.
2. **Model Building**: Implemented classification algorithms such as Logistic Regression, Random Forest Classifier, and Support Vector Machines (SVM).
3. **Evaluation**: Evaluated performance using Accuracy, Precision, Recall, and F1-Score metrics. Handled any class imbalances in the dataset.
