# Restaurant Location-Based Analysis Dashboard

## Overview

This project performs a geographical analysis of restaurants using location, rating, pricing, and cuisine information. A Streamlit dashboard was developed to visualize important business insights and restaurant distribution patterns.

## Objectives

* Analyze restaurant locations using geographical data.
* Identify cities with the highest restaurant concentration.
* Compare average ratings across cities.
* Explore pricing trends by location.
* Discover popular cuisines.
* Generate location-based business insights.

## Features

* Dataset Overview
* Total Restaurants Analysis
* Total Cities Analysis
* Average Rating Analysis
* Most Popular Cuisine Identification
* Top Cities by Restaurant Count
* Key Business Insights Dashboard

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

## Dashboard Metrics

* Total Restaurants
* Total Cities
* Average Rating
* Top Cuisine

## Visualization

### Top Cities by Restaurant Count

Displays the cities with the highest concentration of restaurants.

## Key Insights

* Restaurant distribution is concentrated in a few major cities.
* Customer ratings vary across different locations.
* Premium dining markets have higher average price ranges.
* Certain cuisines dominate the restaurant market.
* Geographical clustering exists within restaurant locations.
