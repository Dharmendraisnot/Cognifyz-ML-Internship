# Task 4: Restaurant Location Analysis

## 🎯 Objective
The goal of this task is to perform spatial data analysis to uncover insights regarding restaurant distributions and regional rating patterns.

## 📈 Steps Performed
1. **Geographical Mapping**: Analyzed the distribution of restaurants across different cities and countries using latitude and longitude coordinates.
2. **Correlation Analysis**: Investigated the relationship between a restaurant's location and its aggregate rating or average cost.
3. **Insights**: Identified popular cuisine clusters and regional hubs with the highest concentrations of top-rated restaurants.
